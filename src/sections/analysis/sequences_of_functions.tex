Let $C \subseteq D \subseteq \R$.

$\bm{\set{f_n}}$ is a sequence of functions each from $D \to \R$.

$\set{f_n}$ \textbf{Converges Pointwise} to $f: C \to \R$ 
$\iff \lall x \in C, \lim_{n \to \infty} f_n(x) = f(x)$. 
Denoted as $f_n \xrightarrow[C]{\textrm{c.p.}} f$,
and $f$ is the \textbf{Pointwise Limit Function}.

$\set{f_n}$ \textbf{Converges Uniformly} to $f: C \to \R$ 
$\iff \lall \e > 0, \lis N = N \in \N, \lall n \geq N, \lall x \in C,
\abs{f_n(x) - f(x)} < \e$.
Denoted as $f_n \xrightarrow[C]{\textrm{c.u.}} f$.

If $\set{f_n}$ Converges Uniformly to $f$, then it Converges Pointwise to $f$.
